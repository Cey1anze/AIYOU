# Install

```
可执行文件已附带程序所需依赖，一键即可启动
```

1. Release中下载最新的版本

2. 双击启动

3. 选择 `Client-Win64-Shipping.exe` 的路径，注意不是WuWa.exe的哦~~

# Build

```
pip install "PyQt-Fluent-Widgets[full]" -i https://pypi.org/simple/
pyinstaller --upx-dir "path/to/your/upx" main.spec
```
